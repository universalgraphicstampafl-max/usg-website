# Showroom v3 — Sketchfab Reference Library

Research basis for the next iteration of the USG interactive 3D showroom (`components/StoreExplorer.tsx`).
Compiled from Preston's reference link + site-wide sweep. All searches filtered to **downloadable** models
(candidates for actual GLTF integration, not just visual reference).

## ⚠️ Licensing rule before using ANY model

The USG site is commercial. Before downloading anything, open the model page and check the license:

- **CC-BY 4.0** → usable commercially WITH attribution (credit the author on the site, e.g. footer of the gallery page)
- **CC0** → usable, no attribution required
- **CC-BY-NC (NonCommercial)** → ❌ CANNOT be used on the USG site
- **"Available on Store"** → paid license via Fab; buy = full commercial rights

## 1. Preston's quality benchmark

**"Modern Store" by Elvair Lima** — sketchfab.com/3d-models/modern-store-7990dc0dd1664fdc9571bff572e411f7
Now sold via Fab (paid). Design elements that make it read as high quality (studied 7/15/26):

- Studio presentation: flat gray backdrop, soft even HDRI light, no harsh sun
- Disciplined palette: tan walls, ONE burnt-orange accent portal, charcoal trims
- Architectural depth: overhanging parapet caps with drip edges, glass INSET into facades with frame returns, mullions with real thickness, beveled edges everywhere
- Glass mostly opaque gray-blue with sheen; interior modeled only behind main glazing
- Baked ambient occlusion at every wall/ground/roof junction
- Restrained landscaping: low shrub clusters in edged planter beds, olive-toned green
- Scale cues: extruded sign letters with glow, door handles, wall lights, furniture

## 9. v3 integration plan

1. **Vet licenses** on shortlist: The Convenience Store, 7-11 convenience store, Gas station (47k),
   Gas station Props, gas pump (7.8k), Supermarket Drinks/Chips shelves, both Blaze light boxes.
2. **Download approved GLTFs** → optimize (gltf-transform / draco) → `public/models/`
3. **Load via drei `useGLTF`** in StoreExplorer, replacing procedural building/pumps/shelving
   where the asset quality beats our boxes; keep our hotspot pin system on top.
4. **Attribution block** for CC-BY assets (gallery page footer or an in-widget credits link).
5. Apply Modern Store design language regardless of asset source: parapet overhangs, inset glazing,
   ContactShadows, muted palette, shrub planters (tracked as v3 polish list).
6. Bundle-size guard: keep total model payload < ~6–8 MB draco-compressed; lazy-load with Suspense.
